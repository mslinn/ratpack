# `Ratpack`

A simple Sinatra app that uses ActiveRecord, Sqlite and Twitter Bootstrap v2.3 (old!). All the other dependencies were updated.


## Up and Running
1. `bundle install`
2. `sudo apt install rerun`
3. `rerun -- rackup --port 4321`
